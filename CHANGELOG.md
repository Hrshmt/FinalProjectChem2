# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-18

### Added
- **Home Page (The Vision)**
  - Hero section with animated headline
  - Floating H₂ molecule background animations
  - Chemistry hook featuring redox equation
  - Feature cards with navigation links
  - Key statistics section

- **The Redox Engine Page**
  - Interactive PEM fuel cell SVG diagram
  - Clickable anode section with oxidation half-reaction
  - Clickable cathode section with reduction half-reaction
  - Clickable membrane and external circuit sections
  - Animated electron and proton flow
  - LEO/GER mnemonic explanations
  - Platinum catalyst section

- **The Gas Lab Page**
  - Kinetic Molecular Theory explanation
  - Hydrogen Pressure Calculator using Boyle's Law
  - Step-by-step calculation breakdown (5 steps)
  - STP values: P₁ = 101.325 kPa, V₂ = 125 L
  - Chemical bonding section with bond energies
  - Animated molecule visualizations

- **The Manitoba Future Page**
  - Manitoba's hydroelectric advantage section
  - Green hydrogen production via electrolysis
  - Cold climate trucking applications
  - Real-world use cases (trucks, transit, agriculture)
  - Environmental significance section

- **Core Components**
  - Glassmorphism navbar with active link indicators
  - Page transition animations with Framer Motion
  - Reusable Card component with hover effects
  - Chemical equation renderer with proper subscripts
  - APA-7 citation system with references

- **Design System**
  - Futuristic dark theme (#050505 background)
  - Neon blue (#00d4ff) and electric green (#39ff14) accents
  - Custom scrollbar styling
  - Smooth scrolling enabled
  - Responsive grid layouts

### Technical
- React 18.3 with Vite 6
- Tailwind CSS v4 for styling
- Framer Motion for animations
- React Router v7 for navigation
- Custom SVG graphics for fuel cell diagram
